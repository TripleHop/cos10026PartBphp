  • Footnotes
    • To cite our information presented more precisely and clearly, footnotes were used. In order to implement this, <sup> were used to create the number in superscript. For it to scroll it to the bottom, where the note is, there needs to be an link. To do this, the superscript was hyperlinked to the note by using the href attribute.
